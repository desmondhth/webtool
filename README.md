# Life Reset Planner

This project is a Vite + React + TypeScript application that renders an interactive Life Reset quiz.

## Running a Production Preview

To build the optimized production bundle and serve it locally using Vite's preview server:

```bash
npm install
npm run build
npm run preview -- --host 0.0.0.0 --port 4173
```

The preview server binds to all interfaces so it can be reached externally if needed. Visit `http://localhost:4173/` in your browser to view the production build.

Press <kbd>Ctrl</kbd> + <kbd>C</kbd> in the terminal to stop the preview server.
